# API Estoque

API REST para gerenciamento de produtos utilizando Node.js, Express, MongoDB Atlas e Swagger.

## Tecnologias

- Node.js
- Express
- Mongoose
- MongoDB Atlas
- Swagger UI

## Instalação

```bash
git clone https://github.com/DiegoZarpok/api-estoque.git
cd api-estoque
npm install

## OBS: Crie um arquivo .env na raiz do projeto e certifique-se de liberar seu IP no MongoDB Atlas em Network Access.

Teste no Swagger: http://localhost:3000/api-docs
